# Análisis de Pacientes con Enfermedad Cardíaca

## Descripción
Análisis exploratorio de datos clínicos de 303 pacientes utilizando Python y Power BI.
Dataset público de UCI Machine Learning Repository (Heart Disease Dataset).

## Herramientas
- Python (pandas, matplotlib, seaborn)
- Google Colab
- Power BI Desktop

## Contenido
- `analisis_pacientes_cardiaca.ipynb` — Notebook con limpieza y análisis exploratorio
- `pacientes_limpios.csv` — Dataset procesado
- `Analisis_Pacientes_Cardiaca.pbix` — Dashboard en Power BI
- `dashboard_pacientes.png` — Captura del dashboard final

## Hallazgos principales
- El 54% de los pacientes presenta enfermedad cardíaca
- Promedio de edad: 54 años
- Los pacientes masculinos representan el 68% del dataset
- El colesterol promedio aumenta con la edad
